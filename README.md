# Fitness-app
Online Fitness Platform built with Angular (frontend) and Spring Boot (backend), focused on designing and implementing a scalable RESTful API.

The system enables users to browse and filter fitness programs with pagination, participate in programs, communicate through comments and messages, and track their personal fitness progress through an activity journal with PDF export and graphical insights.

## Key Backend Features

- RESTful API for managing fitness programs, users, categories, and participation  
- Advanced filtering and search with dynamic attributes per category  
- User authentication with email activation flow  
- Integration with external services (RSS feed and Exercise API)  
- Subscription system with scheduled email notifications  
- Messaging system between users and advisory service  
- Activity tracking with progress analytics

The project also includes:

Admin panel (JSP) for managing categories, users, and system logs
Advisory module (JSP) for handling user inquiries via email responses

Emphasis was placed on clean API design, modular architecture, and separation of concerns between frontend and backend systems
